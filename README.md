## Introduction
Since there is a lot of .io games is so popular in all over the internet. So I try to build a classic snake game. I will design it in 3d and playing it with HTC Cosmos.

## Background
The classic snake game is old but there is still a lot developer develop a brand new style snake game. You can take a look at https://slither.io/. This is game is so popular when it come out. There is still 4000 player per day on the website, some mobile app developer use their API to make player able to play this game where ever you are.

## Method
Try to demonstrate a 3D version snake game that play with computer.
- Design Game Map
Use some package to decorate the map. Make it like more Mario Style.
- Choose a snake package
Just pick a good snake package that look like Mario style.
- Design the hitting object logic
The classic snake game is you score points when you hit an scoring object and your body get longer. So what I want to do is we design it in positive and negative scoring object.
Here is the game logic:
You get longer when you ate a fruit, and you die when you hit the wall, bomb and your own tail.
- Test the game

## Results
I try to do it with 3D angle + HTC Cosmos, but I found out this game will be the best in 2.5D view because I get dizzy when I am play that game with VR gadget.
